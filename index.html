Welcome to the Task Man!

Not another CRUD app

This app was created in the couse of 90minutes with the intent to culminate everything I've learned into one fullstack application. Rather than being a throw away portfolio I have created this with the idea that It may actually help myself and others visualize my work, track progress, and hone my skillset with repition.

Every version/update to this application can be found in the directory above. In this way I can step through my journey revisit old ideas and clearly visualize the course of learning over time. Every commit was created over the course of 90 minutes while each session is created over the course of a day. I should also add that in the spirit of learning, I've organized the directory in sections where each version is added weekly, and updates are sectioned daily with each commit taking no longer than 90 minutes. This serves better as personal reference than a communal functionality. Projects can be found in the Dashboard
